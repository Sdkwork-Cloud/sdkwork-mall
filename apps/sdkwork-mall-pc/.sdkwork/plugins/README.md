# Application Plugins

Application-local agent plugins may be added here for checked-in extensions that apply only to `sdkwork-mall-pc`.

An installable plugin must live at `.sdkwork/plugins/<plugin-name>/.codex-plugin/plugin.json`, use lowercase kebab-case, and document contributed skills, tools, apps, scripts, or MCP servers.

Do not store generated SDK output, secrets, runtime databases, cache, logs, or user-private files in plugins.
