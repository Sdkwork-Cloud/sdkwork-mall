# SDKWork Application Workspace

This `.sdkwork/` directory is source-controlled workspace metadata for the `sdkwork-mall-pc` application root.

It is governed by `../../../sdkwork-specs/SDKWORK_WORKSPACE_SPEC.md` and follows the application entrypoint in `AGENTS.md`.

Authoritative local entries:

- `README.md`: purpose and ownership for this application workspace metadata directory.
- `skills/README.md`: application-local skill contribution guidance.
- `plugins/README.md`: application-local plugin contribution guidance.

Do not store generated SDK transport output, runtime user state, local credentials, secrets, logs, caches, or private files here.
