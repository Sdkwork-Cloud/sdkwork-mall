# Application Skills

Application-local skills may be added here for workflows that apply only to `sdkwork-mall-pc`.

A real skill must live at `.sdkwork/skills/<skill-name>/SKILL.md`, use lowercase kebab-case, cite relevant `../../../sdkwork-specs/` files, and document inputs, touched files, commands, and verification.

Do not store app runtime code, generated SDK output, secrets, local credentials, runtime databases, cache, logs, or user-private files in skills.
